# Naming the authentication tool — search-demand evidence

*Archivist run, 2026-06-30. Goal: name the on-site photo/red-flag tool that helps people check
if a designer bag is real WITHOUT issuing a verdict (markers to check, not a verdict). We pick
the name on evidence, not gut.*

## Honesty preface — what this evidence IS and ISN'T

I do **not** have free-tier access to a keyword tool that returns absolute US monthly search
volumes (Ahrefs/Semrush/Keyword Planner all sit behind logins), and Google Trends returns
**relative** interest, not counts. So this ranking is **directional**, built from three real,
citable signals, and I rank **qualitatively** rather than inventing numbers:

1. **Revealed demand** — which phrasings the resale/authentication industry actually builds its
   pages, tools, app names, and SEO titles around. If everyone titles their page "how to spot a
   fake…", that is the query they've measured traffic on. Strongest signal I can get for free.
2. **Editorial vs comparison vs service split** — the same intent surfaces under different
   phrasings by format (how-to article vs real/fake video vs paid service), which tells us which
   phrasing owns which head.
3. **Community/TikTok usage** — how buyers phrase the request in the wild (captions, forum
   titles). Secondary, but it's the closest thing to "how a person types it."

Every figure/source below carries its URL + a 2026-06-30 check-date. Thin spots are flagged.
No absolute volume is stated as fact anywhere — where I don't have a number, I say so.

## (2) Which name captures the most demand while staying honest (no verdict claim)

**The demand lives in "spot a fake" (rank 1) and "authenticate/authentication" (rank 2) — but
"spot a fake" is the honest, no-verdict winner for a tool name.**

- **"Authenticate / Authentication / Authenticator" (rank 2) captures the most raw demand, but
  it over-promises.** It implies we *render authenticity* — a verdict — which is exactly what
  this tool does not do. Naming a no-verdict tool "[Brand] Authenticator" invites the compliance
  problem the whole "markers to check, not a verdict" rule exists to avoid. Use "authentication"
  in *descriptive body copy and SEO*, not as the tool's promise.
- **"Spot a Fake" (rank 1) is both the highest consumer-demand head AND honest** — "spot"
  = *help you look*, it points at red flags without claiming a determination. It matches how
  buyers actually search and it stays inside the hedge. Recommended tool name direction:
  something built on **"Spot the Fake" / "Fake Spotter" / "Spot-Check"** — a red-flag helper,
  not a verdict engine.
- **Best of both worlds:** name the tool on **"spot a fake"** for honesty + top-of-funnel search,
  and *tag it* with the authentication vocabulary in the metadata/H1 ("… designer bag
  authentication red-flag checker") so we still rank for the rank-2 "authenticate" head without
  making the verdict claim in the name.

**Decisive pick:** lead the name with **"Spot the Fake"** (or "Fake Spotter" / "Real-or-Fake
Spot-Check"). It's the #1 demand phrase, it's verb-honest (look, don't rule), and it dodges the
"authenticator = verdict" trap. Reserve "authentication" for descriptor copy.

## (3) The surprise high-intent phrase to consider naming around

**"Is my [brand] real?" / "how to tell if my bag is real."** This is the sleeper. It's split
across a long tail (so it looks smaller than it is in any single keyword), but it's the
**highest-intent** framing on the list — it's a person with a specific bag *in hand*, ready to
act, which is precisely our tool's user. It also reads as a natural **answer-engine (GEO)**
target: an AI assistant asked "is my Chanel real?" should resolve to our tool. Worth a naming or
tagline nod — e.g. a tagline like *"Is it real? Let's check the markers,"* which captures the
query, states the value, and keeps the hedge (check the markers, not "we'll tell you if it's
real").

Secondary surprise: **"authenticity check"** shows up as an organic *community caption* more
than as a search term — useful as a familiar on-page label ("run an authenticity check"), even
though it's a weak demand magnet on its own.
