## Working title

**Which luxury houses actually name their bag colors (and which don't)**

## The hook (de-gatekeeping, one paragraph)

People ask "what is my bag's color called" expecting every house to have an answer. Some do, most
don't, and that is the actually useful thing to know. A few houses run a real proprietary color
lexicon, where a shade has an official house name (and sometimes a code) that follows the bag for
its whole life. Most houses don't: the color on the tag is a plain descriptor like "Black" or
"Tan," picked per season, with no house dictionary behind it. One house sits in between. Here is
where each one lands, so the next time someone tells you a bag is "a rare Etoupe" or "just camel,"
you know which is a real house name and which is a description someone wrote.

## The three camps

### Camp 1: A real proprietary color lexicon (the name is the house's own)

**Hermès.** Hermès names every color and assigns it an official numeric code, and the code is the
anchor. The confirmed set includes **Noir 89** (black), **Gold / Or 06** (warm camel), **Rouge H 46**
(deep brownish red, in the house since 1925), **Etoupe 18** (the bestselling grey-brown taupe),
and **Orange H / Feu 93** (the signature box orange). So "Etoupe" is not a description of a taupe
bag; it is the color's actual house name, and 18 is its code. (Source: `hermes.md` §2.)

Honest caveat to keep in the piece: many Hermès colors beyond that confirmed set have a name but no
code sourced yet (Gris Tourterelle, Nata, Beton and others are marked "color confirmed, code not
yet sourced" in the archive). So name the codes that are confirmed, and say plainly that the rest
are colors with names whose codes are still being sourced, rather than inventing a number.

**Bottega Veneta.** Bottega is the other house that genuinely names its colors, which makes it the
counter-case to most of Italian luxury. The fully confirmed official one is **Parakeet**, the
grassy acid green Daniel Lee reintroduced for Spring/Summer 2021, now part of the brand's store
design and branding. (Source: `bottega-veneta.md` §3, double-sourced.)

The hedge that matters: Bottega's wider color list (Fondant, Porridge, Barolo, Travertine, the
"Washed" pastels and the rest) comes from how a major reseller, Fashionphile, catalogs the colors,
not yet confirmed against bottegaveneta.com. These are very likely Bottega's own names, but the
archive marks them "descriptive, reseller-attributed" pending a house confirm. So in the piece:
**Parakeet is the one safe-to-state official Bottega color**; the rest are the names collectors and
resellers use, a strong lead, not yet house-verified. (Source: `bottega-veneta.md`, "Official vs
descriptive" rule + §3.)

### Camp 2: No house color lexicon (the color is a plain descriptor)

These houses identify a bag by model, material, or a season/style code, and leave the color as an
ordinary description picked per season. There is no house dictionary that says "this season's pink
is called X."

- **Dior.** Dior does not publish a unique official name per Lady Dior seasonal colorway. Its named
  layer is capsules and numbered art editions (Dior Lady Art, My ABCDior, Toile de Jouy), not
  colors. (Source: `dior.md` Hard Rule 3.)
- **Gucci.** GG Marmont and Dionysus rotate plain color descriptors with no per-season house name.
  The one genuine exception is **Rosso Ancora**, the deep oxblood red Sabato De Sarno named for his
  2023 chapter (SS2024 debut). One named color, not a lexicon. (Source: `gucci.md` Hard Rule 3 + §3.)
- **Saint Laurent.** YSL leaves its colors as plain descriptors (Noir, Creme, Dark Beige, Rouge).
  The "name" that carries weight on a YSL bag is the material and the hardware tone, not the color.
  (Source: `saint-laurent.md`.)
- **Celine.** Tan, Black, Camel, Natural, White: plain descriptors, no poetic seasonal-color
  lexicon. (Source: `celine.md`.)
- **Fendi.** Brown, Black, Camelia and the like are plain descriptors; the naming weight sits on the
  logo canvas and the leather line, not color. (Source: `fendi.md`.)
- **Prada.** Identified by model, material (Saffiano vs nylon), and the triangle plaque, with colors
  as plain descriptors, often flat Italian shade words (Rosa = pink, Cammeo = a nude beige, Talco =
  off-white). No per-season color-name dictionary. (Source: `prada.md` §3.)
- **Chanel.** Chanel ships a style code and a season code (like 18C) and leaves the seasonal color
  unnamed. So "what pink was Cruise 2018" has no official house answer, even though the season code
  itself is a real, decodable Chanel name. (Source: `chanel.md`, via `bottega-veneta.md` cross-house
  context; and content-ideas.md, the "Chanel color trap" note.)

### Camp 3: The middle (a seasonal list, mostly descriptive)

**Loewe.** Loewe is the honest in-between case. It does publish a per-season color list, which is
more than the Camp 2 houses do: its own SS24 Flamenco page names four seasonal colors, **Emerald
Green, Dark Burgundy, Sahara, and Black**. But the names are mostly literal (Emerald Green, Black),
with only the occasional evocative one (Sahara). So Loewe labels its seasonal colorways, but it is
not running the deep proprietary lexicon Hermès and Bottega do. (Source: `loewe.md` §3, loewe.com
SS24 list.)

## Why this answers a real question

This is why "what is my bag's color called" has a real answer for some houses and not others. For an
Hermès bag, the color genuinely has a house name and usually a code, so the answer is exact. For a
Bottega, Parakeet is exact and the rest are the names the resale market uses. For a Dior, Gucci,
Saint Laurent, Celine, Fendi, Prada, or Chanel bag, the honest answer is that there is no official
color name to give: the bag is identified by its model, material, or season code, and the color is
just a description. Knowing which house you are holding tells you which kind of answer to expect.
