# Research: How to authenticate a Louis Vuitton bag (#4)

*Sourced 2026-06-26. The verified basis for the LV authentication article. Bound by
`docs/authentication-standard.md` (markers to check, never a verdict; never assert what we
can't source; no confidence badge; original brand-neutral schematic, NO reproduced LV
monogram/Damier/logo). Pairs with the Coach guide's structure.*

## The hook: date codes are gone (2021)
- Louis Vuitton **stopped putting visible date codes in bags around March 2021** and replaced
  them with a **hidden NFC/RFID microchip** embedded in the lining. Sources: Real
  Authentication, PhilipKarto, Luxity, Lollipuff, Pro Authenticators.
- A date code was **never a serial number**: it encoded the factory + the production date, and
  many bags shared the same code. Counterfeiters learned the formats, which is *why* LV moved
  to the chip.
- **The chip is not a DIY check.** Only Louis Vuitton can read the encrypted data; professional
  authenticators can confirm a chip is present in the expected spot but cannot read it. So the
  chip does not let a buyer self-verify.
- **Do NOT publish a date-code decoder** (factory letters / per-year date formats). Per the
  auth-standard never-invent rule (and the Coach precedent of not asserting per-year tables):
  explain that the code exists and what it broadly is, that it is **not proof** (fakes copy
  codes), and stop there.

## Markers to check (sourced; all "check," none a verdict)
- **Heat stamp** ("LOUIS VUITTON ® made in [country]" pressed into a leather tab): deep, crisp,
  even press. Sourced font tells: the **O in Vuitton is round, not oval**; the **TT nearly
  touch**; the **® is small with breathing room** (doesn't touch the L or the stitching). Frame
  carefully: these are tells counterfeiters often miss, but **good fakes replicate them**.
  Source: Fashionphile Academy, Legitique, Luxury Columnist.
- **Hardware:** solid brass, **noticeably heavy**, not hollow/light; engravings on zipper pulls,
  rivets, clasps are clean and sharp. Source: Borro, Luxury Columnist.
- **Stitching:** even, **mustard/yellow waxed thread**, high stitch count (dense, not sparse
  with gaps), consistent ~45° angle on tabs; handle stitching is short and compact. Source:
  Fashionphile Academy, Legitique.
- **Vachetta leather:** the untreated tan leather patinas honey-to-brown over time; a brand-new
  "vintage" bag with pale flawless vachetta is a flag (general; state as a leaning).
- **Canvas alignment / symmetry:** the monogram or Damier should run **symmetrically** across
  seams (we describe this in words; the diagram must NOT reproduce the pattern).

## Red flags (sourced)
- **An "authenticity card."** LV does **not** include authenticity cards; one is a flag, not
  proof of realness. Source: multiple guides.
- Missing/absent heat stamp; sloppy or shallow embossing; light/hollow hardware; price too good
  to be true.

## The honest endpoint (auth-standard §, mandatory)
No single marker proves authenticity; a good fake passes visual checks; the microchip can't be
read by a buyer. Before a costly purchase, or to sell/insure, **send it to a professional
authenticator.** Markers to check, not a verdict.
