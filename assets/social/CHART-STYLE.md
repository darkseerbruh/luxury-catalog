# House chart style (locked 2026-06-30)

The look for every data-viz pin/graphic. Luxe and professional underneath, one human,
un-gatekept touch on top. Frameless. Pinterest 2:3 (viewBox 1000x1500).

## Palette (jewel base, default = plum)
- Background: vertical gradient `#311525` → `#180a11` (plum). Alternates allowed: deep green
  (`#1b2e25`→`#0e1813`) or charcoal (`#14120c`). Pick one per piece; do not mix.
- Gold (primary data / accent): `#c9a24c`. Muted bar (the cheaper / secondary): `#8a6f63`.
- Headline cream: `#f3e9ec`. Body/labels muted: `#b9a6ad`. Footer/hedge: `#9a838c`.
- Baseline rule: `#5a3a46`.
- Type: Georgia serif throughout. Kicker = small, letter-spaced, gold, all-caps.

## Structure (top → bottom)
1. Kicker (letter-spaced gold), then a two-line serif headline (second line gold italic).
2. One-line subtitle: "<model>, median asking/sold by size, June 2026".
3. The chart: bars off a thin baseline. Highlight the point bar in gold, others muted.
4. Value labels in cream above each bar.
5. **Bag silhouettes (single-model size charts only):** generic, NON-LOGO structured-bag
   icon (`#bag` symbol), scaled up across the sizes so the bag grows while the price falls.
   Skip on comparison charts (two models / ask-vs-sold).
6. A warm italic tagline ("Same bag. More leather. Less money.").
7. Footer hedge, always: "Asking/sold medians, June 2026. An estimate of the market, not an
   appraisal. luxurycatalog.com".

## Rules
- **No border frame** (it tries too hard).
- **No on-image caption + no hand-drawn arrow** (tried, cut, the bags carry the story).
- **n stays off the social pins** (it lives in the on-site article charts; pins read clean).
- Give labels room (don't clip tops). Every number traces to our dated capture.
